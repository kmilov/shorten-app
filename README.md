# Shorten URL APP
A React/Redux application to shorten an URL using the API
gymia-shorty.herokuapp.com/

The app has been tested in Chrome 56+

## How to run
```
$ git clone https://github.com/kmilov/shorten-app.git
$ cd shorten-app
$ npm install
$ npm start
navigate to http://localhost:8000
```
